# Projetos HTML e CSS — Flexbox e Grid

---

## Sobre

Este repositório reúne 3 projetos desenvolvidos durante meus estudos de HTML e CSS em um bootcamp de desenvolvimento web. Os projetos foram criados para praticar Flexbox, CSS Grid e Responsividade, focando na organização de layouts modernos e na estruturação de páginas com HTML semântico.

---

## Projetos Desenvolvidos

### Projeto 1 — Lendas da Seleção Brasileira
Portal inspirado em jogadores históricos da Seleção Brasileira, como Pelé, Romário e Ronaldo Fenômeno.

*   **Conceitos praticados:**
    *   `display: flex`
    *   `justify-content`
    *   `align-items`
    *   `flex-wrap`
    *   Sidebar fixa
    *   Responsividade

### Projeto 2 — Como Vencer Recrutadores
Portal focado em carreira e tecnologia com dicas sobre Currículo, LinkedIn, Entrevistas e Networking.

*   **Conceitos praticados:**
    *   `display: grid`
    *   `grid-template-columns`
    *   Grid dentro de Grid
    *   `repeat(auto-fit, minmax())`
    *   Responsividade

### Projeto 3 — Tecnologia em 2026
Portal sobre tendências tecnológicas e o futuro da TI, contando com seções de Inteligência Artificial, Cloud Computing, Cyber Segurança, Desenvolvimento Web e Carreiras em Tecnologia.

*   **Conceitos praticados:**
    *   `grid-template-areas`
    *   Layout responsivo
    *   Áreas nomeadas
    *   Estruturação avançada com Grid

---

## Aprendizados

Durante o desenvolvimento destes projetos, foi possível compreender:

*   A diferença prática entre Flexbox e Grid.
*   Como estruturar layouts responsivos.
*   Quando usar cada abordagem de estilização.
*   Organização de páginas utilizando HTML semântico.
*   Estruturação visual com CSS avançado.
